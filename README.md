# Intellij IDEA/PyCharm Community 简体中文语言体验包存档

## Purpose

由于 JetBrains 在 2025.* or later 的 IDE 中不再在 Marketplace 中提供单独的语言包，而在 GitHub 中仍然有 Intellij/PyCharm Community 的构建，且 PyCharm Community 的安装包体积是 Unified (统一产品) 的一半左右，故使用此仓库存储语言包资源。

## 资源分支
